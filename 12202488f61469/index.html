变量命令规则

  1. 数字名称区分大小写
  2. 标记名长度不超过63位ABCDEFGHIJKLMNOPQRSTUVWXYZ123456ABCDEFGHIJKLMNOPQRSTUVWXYZ123456 = 3
  3. 标志名以字母开头，可以由字母、数字和下划线组成，但不能使用标志点 3A = 4 .a = 5 /b = 5
  4. 指标名称应简洁明了，通过指标名称可以理解出指标所表示的物理意义

绘图

二维绘图

x = 0:0.01:2*pi;
y = sin(x);
figure%绘制一个幕布
plot(x, y)%绘制x轴和y轴
title('y = sin(x)')%标题
xlabel('x')%x轴标签
ylabel('sin(x)')%y轴标签
xlim([0 2*pi])%限制x轴方向图像长度

x = 0:0.01:20;
y1 = 200*exp(-0.05*x).*sin(x);
y2 = 0.8*exp(-0.5*x).*sin(10*x);
figure
%共用一个x坐标在y上有不同的取值
[AX,H1,H2] = plotyy(x,y1,x,y2,'plot');
%doc set
set(get(AX(1),'Ylabel'),'String','Slow Decay')
set(get(AX(2),'Ylabel'),'String','Fast Decay')
xlabel('Time (\musec)')
title('Multiple Decay Rates')
set(H1,'LineStyle','--')
set(H2,'LineStyle',':')

  1. txt格式
M = importdata('myfile.txt');

S = M.data;
save 'data.txt' S -ascii
T = load('data.txt');

isequal(S, T)
  1. xls格式
xlswrite('data.xls',S)
W = xlsread('data.xls');
isequal(S, W)

xlswrite('data.xlsx',S)

csvwrite('data.csv',S)
